URL CLEANER

Rules

How to write rules

One rule per line. By default, when a matching parameter is found, that parameter and everything after it in the URL is removed. Add ~ suffix (e.g., ?param~) to only remove that single parameter.

  • ?param, &param, or #param - Remove parameter and everything after it from all sites
  • ?param~, &param~, or #param~ - Remove only this single parameter from all sites
  • domain.com:param - Remove parameter and everything after it from specific domain
  • domain.com:param~ - Remove only this single parameter from specific domain
  • !domain.com - Whitelist domain (never clean URLs from this domain)
  • !domain.com:param - Do NOT remove this parameter from this domain
  • !param - Never remove this parameter from any domain
  • &param* - Wildcard matching (e.g., &utm_source* matches &utm_source_extra)

Lines beginning with // are comments, and blank lines are skipped.
